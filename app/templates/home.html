Este projeto foi desenvolvido com foco no aprendizado e nas boas práticas de desenvolvimento web, utilizando:

Flask (Python): Framework web leve e poderoso, que organiza toda a estrutura da aplicação.

MySQL: Banco de dados relacional onde ficam armazenadas as informações dos usuários e tarefas.

SQLAlchemy: ORM (Object-Relational Mapper) que facilita a comunicação com o banco de dados através de objetos Python.

Flask-Login: Biblioteca usada para autenticação de usuários e gerenciamento de sessões com segurança.

Jinja2: Template engine do Flask, responsável por exibir as informações de forma dinâmica nas páginas HTML.

HTML e CSS personalizados: Para uma interface clara, responsiva e funcional.

Javascript

  • Atualiza o DOM dinamicamente (por exemplo, ao adicionar ou remover uma tarefa);
  • Realiza requisições HTTP assíncronas (AJAX) para a API do Flask usando fetch, melhorando a fluidez da experiência do usuário;
  • Manipula eventos de forma interativa (ex: cliques, envio de formulários, etc).
